# ShipSwift - Internal Package Tracking System

# Description

ShipSwift is a web-based internal package tracking system used to monitor shipments, track delivery status, and manage logistics efficiently.

# Features

* Dashboard with package statistics
* Package tracking system
* Shipment creation
* Alerts & exception handling
* Analytics dashboard
* Live route tracking (map)

 **Technologies Used**
* HTML
* CSS
* JavaScript
* Leaflet.js

# How to Run

1. Download or clone this repository
2. Open `index.html` in your browser
